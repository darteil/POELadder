# POE Ladder

**Electron application for viewing the Path of Exile ladder.**

![](http://darteil-projects.ru/static/images/robot-civ-clicker.png)

**To Use.**

```bash
# Install dependencies
npm install

# Run the app
npm run prod
npm start

# Run in browser
npm run dev

# Create release ("release" folder)
npm run pack
```
