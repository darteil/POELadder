# POE Ladder

**Electron application for viewing the Path of Exile ladder.**

![](public/Screenshot.png?raw=true)

**To Use.**

```bash
# Install dependencies
npm install

# Run the app
npm run prod
npm start

# Run in browser
npm run dev

# Create portable exe ("release" folder)
npm run pack
```
